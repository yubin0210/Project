# ITving
***
JSP를 활용한 CRUD사이트 개발 프로젝트. 기존 Tving이라는 사이트의 디자인을 모티브로 제작하였습니다.
***
### 개발인원

5명


### 개발기간

2024.01 ~ 2024.02 (약 3주)


### 개발환경

**Back-End** : Java

**Framework** : JSP

**DataBase** : Oracle SQL

**Front-End** : HTML, CSS
